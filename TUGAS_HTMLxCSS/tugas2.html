Berikut cerita perjalanan hidup saya

Saya berasal dari kalimantan barat tepatnya di pontianak, kemudian saya pindah ke kota lain bernama sintang kemudian saya TK negeri kemudian SD,SMP,dan SMA saya masuk ke sekolah swasta. Saya pertama kali mengenal komputer itu saat di pontianak tetapi itu hanya sekedar mengetahui tidak tau cara menjalankannya kemudian saya bisa menggunakan komputer itu saat disintang saat saya sudah masuk TK, saat itu saya menggunakan laptop orang tua hanya untuk bermain game. Kemudian saat saya masuk SD disitu saya baru benar-benar belajar mengunakan komputer tetapi pembelajaran yang sederhana saja seperti belajar word. Bagaimana cara menulis,membuat baris,memilih font serta ukuran dan juga belajar shape. Di tempat saya pembelajaran komputernya tidak terlalu maju jadi hanya belajar mengetik saja di word. Excel ada tetapi hanya sebentar saja. Belajar excelpun itu waktu SMA aja dan hanya excel-excel dasar.
